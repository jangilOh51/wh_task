# 프로젝트 개요
이 프로젝트는 API 테스트 자동화 구현 및 테스트 케이스 문서에 대한 내용입니다.

- **`api_test`**: `pytest` 코드를 테스트를 실행하고, 관리하는 테스트 프레임워크입니다.

# 주요 지침
- 모든 답변은 한글로 답변해주세요.
- git에 commit할때도 한글로 comment를 입력해줘 이해할 수 쉽게.

# 주요 라이브러리
- poetry : 코드 환경
- python3 : 코드 언어
- pytest : 테스트 프레임워크
- allure : 리포트
- requests : api 호출

# 실행 방법
- **테스트 코드 실행:**
  ```bash
./scripts/run_tests.sh
  ```

```bash
./scripts/run_tests.bat
  ```
